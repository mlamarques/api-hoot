# STEPS

### Start the package.json file: <code>npm init -y</code>

### install express(server), mangoose(db) and passport(middleware for authentication)

```
npm install express express-session mongoose passport passport-local
```

### Install dotenv

```
npm i bcrypt
```

### Install bcrypt
